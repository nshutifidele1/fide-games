# **App Name**: Fide Games

## Core Features:

- Immersive Hero Scene: A dynamically animated 3D hero section showcasing a central gamer character, futuristic weaponry, background elements like drones, holograms, and animated atmospheric effects (smoke, particles, light streaks) integrated with Three.js.
- Dynamic Navigation & Auth: A transparent, sticky navigation bar with main menu links and interactive 'Login' / 'Sign Up' buttons featuring neon hover animations and glowing effects.
- Featured Games Display: An interactive carousel presenting large game cards with a glassmorphism style, basic cover image, rating, genre, and a 'Play' button, including hover tilt animation and glowing shadows.
- AI Game Discoverer Tool: An AI-powered tool that analyzes user browsing and interaction patterns to suggest personalized game recommendations within dedicated sections.
- Community Reviews & Ratings: A dedicated section displaying game review cards, including a neon circular rating meter with animated counters to highlight community scores.
- Trending Gaming News Feed: A section showcasing trending news with cinematic card layouts, animated thumbnails, parallax hover effects, and 'Read More' neon buttons.
- Global Interactive 3D Effects: Implementation of subtle, mouse-reactive lighting and floating particle effects across various sections to enhance the 3D aesthetic and user engagement, integrated using Three.js.

## Style Guidelines:

- Primary Color: A vibrant violet (HSL(270, 80%, 50%), equivalent to #6619E5) chosen to embody the deep, mystical and cutting-edge aspects of the cyberpunk genre, forming a strong base for the UI's energetic elements.
- Background Color: A deeply desaturated, dark purple-grey (HSL(270, 20%, 12%), equivalent to #1F1825) providing a stark, dark canvas that allows neon elements to pop and ensures high contrast for readability in a dark mode environment.
- Accent Color: An electric blue (HSL(240, 90%, 65%), equivalent to #5555F6) used to highlight interactive elements, calls to action, and subtle glowing borders, offering a dynamic contrast to the primary violet and reinforcing the neon aesthetic.
- Headlines and prominent text: 'Space Grotesk' (sans-serif), for a sharp, techy, and futuristic feel fitting for game titles and major sections. Body text and detailed descriptions: 'Inter' (sans-serif), chosen for its modern readability and neutral character, ensuring legibility for longer blocks of content like reviews and news articles.
- Futuristic and geometrically precise neon-style icons for navigation, game categories, and social media links, featuring subtle glow effects to maintain the cyberpunk aesthetic.
- Leverage a full-screen, responsive layout with transparent sticky navigation, generous spacing, glassmorphism-inspired UI cards, and cinematic presentations for featured content, aiming for a AAA gaming platform feel.
- Extensive use of smooth 3D animations and transitions throughout: animated hero sections, holographic floating UI elements, glowing borders and interactive neon hover states, parallax scrolling, mouse-reactive lighting, and particle effects to create an 'alive' and immersive experience, implemented with Framer Motion and Three.js.
